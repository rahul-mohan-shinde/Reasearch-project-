# Reasearch-project
Rahul, अगर तुम्हारा **MCA Research Project (RP641RP)** है और तुम्हें **100/100 या Out of Marks** चाहिए, तो सिर्फ रिपोर्ट बनाना काफी नहीं होगा। SPPU के दिए हुए guidelines के हिसाब से पूरा काम करना होगा।

## 📌 1. Marks Distribution (100 Marks)

| Criteria                         | Marks   |
| -------------------------------- | ------- |
| Originality of Proposed Work     | 10      |
| Literature Survey                | 10      |
| Proposed Work                    | 10      |
| Results Obtained                 | 10      |
| Detailed Report                  | 30      |
| Paper Presentation / Publication | 20      |
| Final Presentation (Viva + PPT)  | 10      |
| **Total**                        | **100** |

---

# 🎯 Out of Marks लाने के लिए क्या करना होगा?

## 1. Original Topic (10 Marks)

ऐसा topic लो जिसमें कुछ नया हो।

### Example:

* AI Based Hospital Management System
* Smart Appointment Booking System
* WorkNest SaaS Platform
* AI Resume Analyzer
* Medical Recommendation System

---

## 2. Literature Survey (10 Marks)

कम से कम **25 Research Papers (2020 के बाद)** पढ़ने होंगे।

Sources:

* IEEE
* Google Scholar
* Kaggle
* GitHub
* Scopus

इनसे निकालना है:

* Author
* Year
* Method Used
* Advantages
* Limitations

फिर comparison table बनानी है।

---

# Formatting Rules

✅ Font: Times New Roman

✅ Size:

* 12 pt (Body)
* 14 pt (Headings)

✅ Line Spacing:

* 1.5

✅ Alignment:

* Justified

✅ References:

APA Style

✅ Plagiarism:

* Text ≤10%
* Code ≤15%
